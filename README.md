# Basic Chatbot 🤖

## Description
A simple rule-based chatbot built using Python. The chatbot responds to predefined user messages and continues the conversation until the user types "bye".

## Features
- Responds to common greetings
- Answers basic questions
- Uses conditional statements (if-elif-else)
- Interactive command-line interface
- Exit option using "bye"

## Technologies Used
- Python
- Loops
- Conditional Statements
- User Input/Output

## Project Structure

BasicChatbot/

## How to Run

1. Clone the repository or download the files.
2. Open a terminal in the project folder.
3. Run the program:

```bash
python basic_chatbot.py
```

## Sample Interaction

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: what is your name
Bot: I am a Python chatbot.

You: bye
Bot: Goodbye!
